# 📊 NLP: Sentiment Analysis on Movie Reviews

## 👋 About Me
Hi, I'm Suraj Singh, a B.Tech graduate aspiring Data Science Engineer.

## 📌 Project Overview
This project focuses on building a sentiment analysis model to classify Amazon movie reviews as positive or negative using Natural Language Processing (NLP). The goal is to understand text data and predict sentiment accurately.

## 🎯 Objectives

- The AmazonReviews.TSV File On Kaggle
- Link oF The File : https://www.kaggle.com/datasets/surajsingh83/amazonreviews-tsv
  
- Perform text cleaning and preprocessing  
- Convert text data into numerical features  
- Build classification models for sentiment analysis  
- Evaluate model performance using appropriate metrics  

## 🧪 Methodology
- Data collection and loading (TSV format)  
- Text preprocessing:
  - Lowercasing  
  - Tokenization  
  - Stopword removal  
  - Stemming / Lemmatization
    
- Feature extraction:
  - TF-IDF Vectorization  
- Model building:
  - Logistic Regression  
  - Naive Bayes  
- Model evaluation using metrics:
  - Accuracy, Precision, Recall, F1-score  

## 📊 Models Used
